# CashFlowDjango

**CashFlowDjango** — веб-приложение для управления движением денежных средств.

Приложение позволяет пользователям:
- создавать новые записи о денежных операциях;
- просматривать историю движения денежных средств;
- редактировать существующие записи;
- удалять операции;
- управлять справочниками и данными через административную панель Django.

---

## Установка и запуск

### 1. Подготовка окружения

Убедитесь, что у вас установлен **Python 3.14**.

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте его:

**Windows:**
```bash
venv\Scripts\activate
```

**Linux / macOS:**
```bash
source venv/bin/activate
```

---

### 2. Установка зависимостей

Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

---

### 3. Настройка базы данных

Создайте миграции:

```bash
python manage.py makemigrations
```

Примените миграции:

```bash
python manage.py migrate
```

---

### 4. Заполнение тестовыми данными (опционально)

Для добавления тестовых записей выполните:

```bash
python manage.py shell < db_seed.py
```

---

### 5. Создание администратора

Для доступа к административной панели Django создайте суперпользователя:

```bash
python manage.py createsuperuser
```

Введите:
- имя пользователя;
- почту можно пропустить
- пароль;

После создания администратора вы сможете управлять записями и справочниками через панель или по кнопкам:

```
http://127.0.0.1:8000/admin/
```

---

### 6. Запуск приложения

Запустите сервер разработки:

```bash
python manage.py runserver
```

Откройте приложение в браузере:

```
http://127.0.0.1:8000/
```

---

## Авторизация

Для выполнения операций:
- добавления записей;
- редактирования;
- удаления;

необходимо ввести учетные данные зарегистрированного пользователя.
